This is another trial repository, trying to master github. Step1: Fork the repository from the fork icon on the top-right. Step2: Clone the repository to work on it in the locl system. This can be done by copying the repository link form the code option, and then uginf the git command 'git clone' to clone it. Step3: After clonign, open the repo from C-Windows-Users folder in the editor and do the required changes. Step4: After the changes are done, open the git options in the menu and push the changes. Step5: After the changes are pushed, commit them and finally send a pull request.
